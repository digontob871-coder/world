# 🏆 World Cup Live Score 2026

A comprehensive real-time live scoring platform for the FIFA World Cup 2026 with detailed player statistics, group standings, tournament information, and AI-powered insights.

## ✨ Features

✅ **Live Match Scores** - Real-time match updates with live scores  
✅ **Group Stage & Knockout Rounds** - Complete tournament structure from group phase to final  
✅ **Stadium Information** - Venue details and locations across USA, Mexico, Canada  
✅ **Player Tracking** - Individual player performance ratings  
✅ **Group Standings** - Dynamic points calculation  
✅ **Historical Data** - Past World Cup winners and records  
✅ **Player Statistics** - Goals, assists, Ballon d'Or, UEFA Champions League, World Cup wins  
✅ **Multi-Language Support** - English and multiple languages  
✅ **AI Assistant** - Google Gemini powered chatbot for queries  
✅ **Countdown Timer** - Days until World Cup 2026  
✅ **Sofascore Theme** - Modern UI similar to Sofascore  

## 🛠️ Tech Stack

- **Frontend**: React.js, TypeScript, Tailwind CSS
- **Backend**: Node.js, Express, MongoDB
- **APIs**: 
  - API-Sports (Football data)
  - Sofascore API (Live scores)
  - Google Gemini (AI Assistant)
- **Database**: MongoDB
- **Deployment**: Vercel / AWS
